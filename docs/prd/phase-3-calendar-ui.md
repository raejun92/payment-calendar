# Phase 3: 커스텀 달력 UI PRD

## 목표
결제 내역을 날짜별로 확인할 수 있는 달력 화면을 구현한다.

## 배경
- 달력은 앱의 메인 화면으로, 사용자가 가장 많이 보는 화면
- 한눈에 일별 결제 금액을 파악하고, 상세 내역을 빠르게 확인할 수 있어야 함
- 4단계(데이터 모델 + CRUD)에서 실제 데이터를 연결하기 전, UI를 먼저 완성

## 범위

### 1. 달력 라이브러리
- `react-native-calendars` 사용
- 월간 달력 뷰
- 좌우 스와이프로 월 이동
- 오늘 날짜 강조 표시

### 2. 달력 레이아웃
```
┌──────────────────────────────┐
│  2026년 3월        ₩ 352,000 │  ← 월/년 + 월별 합계
├──────────────────────────────┤
│  일  월  화  수  목  금  토   │
│                              │
│          1     2     3    4  │
│               5,000        │
│                              │
│  ...                         │
│                              │
│  ⦿30                        │  ← 오늘 날짜 강조
│   12,000                     │
├──────────────────────────────┤
│  3월 30일 (월)               │  ← 선택한 날짜 상세
│                              │
│  14:30  홍 스타벅스   ₩5,000  │  ← 시간순 위→아래, 결제자 이니셜
│  18:20  김 쿠팡      ₩7,000  │
│                              │
└──────────────────────────────┘
```

### 3. 금액 표시 형식
- 통일 형식: `₩5,000` (원화 기호 + 천 단위 콤마)
- 날짜 셀: `₩5,000` (총액)
- 하단 리스트: `₩5,000` (개별 금액)
- 월별 합계: `₩352,000`

### 4. 날짜 셀 표시
- 날짜 숫자 아래에 해당 날짜의 **결제 총액** 표시
- 결제가 없는 날짜는 금액 미표시
- 스타일 우선순위:
  - **선택 날짜**: 채워진 원형 배경 (primary color)
  - **오늘**: 테두리 원형 (보조)
  - **오늘 + 선택**: 채워진 원형 (선택이 우선)

### 5. 날짜 선택 시 하단 리스트
- 달력 높이는 고정, 하단 리스트만 스크롤
- 선택한 날짜의 결제 내역을 달력 아래에 리스트로 표시
- 시간순 정렬 (위 → 아래 = 오래된 순 → 최신)
- 각 항목: 결제자 이니셜(성), 시각, 가맹점명(있으면), 금액
- 결제자 구분: 이니셜 + 색상으로 구분 (예: 홍=파랑, 김=초록)
- 결제 내역이 없으면 "결제 내역이 없습니다" 표시
- 앱 첫 진입 시 기본으로 오늘 날짜 선택

### 6. 월별 합계
- 달력 상단 헤더 영역에 현재 월의 총 결제 금액 표시
- 월 이동 시 해당 월의 합계로 업데이트
- 월 이동 시 선택 날짜: 오늘이 해당 월에 포함되면 오늘 선택, 아니면 1일 선택

### 7. 더미 데이터
- 실제 Firestore 연동은 4단계에서 진행
- 이 단계에서는 **더미 데이터**로 UI를 구현하고 검증
- 기존 `types/payment.ts`의 `Payment` 타입을 그대로 사용
- 타임존: **KST (Asia/Seoul)** 기준으로 날짜 그룹핑
- 정렬: `time` 오름차순 (오래된 순 → 최신)
- 더미 데이터에 결제자(userId, displayName) 정보 포함

### 8. 프로젝트 구조 추가
```
components/
  └── calendar/
      ├── calendar-header.tsx     # 월/년 표시 + 월별 합계
      ├── calendar-day.tsx        # 날짜 셀 (금액 표시)
      └── payment-list.tsx        # 선택 날짜의 결제 내역 리스트
constants/
  └── dummy-data.ts               # 더미 결제 데이터 (4단계에서 제거)
```

## 이 단계에서 하지 않는 것
- Firestore 실시간 데이터 연동 (4단계)
- 결제 추가/삭제 기능 (4단계)
- 주간 뷰
